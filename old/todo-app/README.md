# Todo app example

## Usage

1. Run

   ```bash
   npm run env:init
   ```

2. Fill out `.env.prod` and `.env.dev` with parameters

3. Run this to set development environment variables
   ```bash
   npm run env:dev
   ```

4. Run this to start in development (watch and rebuild) mode
   ```bash
   npm run start:dev
   ```

### To use it in production
1. Run
   ```bash
   npm run env:prod
   ```

2. Run
   ```bash
   npm run build
   ```

3. Run
   ```bash
   npm run start
   ```
